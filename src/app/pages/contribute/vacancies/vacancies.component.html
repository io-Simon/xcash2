Project Manager

Missions

We are looking to onboard a contributor with project management skills to develop the XCASH contributor program and help us supervise it.

As the Project Manager, you will be responsible for:

-         Finalizing the contribution guidelines with the other contributors

-         Define the contribution workflow in GitHub Project

-         Follow up and update the existing pipeline of projects

- Apply and enforce the contribution guidelines for healthy repositories

-         Attend the contributor calls and provide updates to the rest of contributors

-         Coordinate with the communication team to provide articles and content to update the community

 

Contributor Profile

-         Previous relevant experiences in project management

-         Knowledge of the XCASH Core technology and principles

-         Understanding of GitHub workflow and GitHub Projects

-         Great communication and written skills (English)

-         Proactive and Curious

-         Being already active in the XCASH community is a strong plus
